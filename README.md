# Generatore Firma Email - Comune di Vallelaghi

Applicazione web semplice per generare la firma email standardizzata per i dipendenti del Comune di Vallelaghi.

ðŸ”— **[Usa l'applicazione online](https://sergiopovoli.github.io/email_sign/)**

## Come usare

1. Apri `index.html` nel browser
2. Compila il form con i tuoi dati:
   - Nome
   - Cognome
   - Ruolo
   - Ufficio/Altro (opzionale)
3. Visualizza l'anteprima della firma
4. Clicca su "Copia Firma"
5. Incolla la firma nelle impostazioni di Gmail
